> **Full-Stack AI SaaS** — Speak or type in Egyptian Arabic — AI books the appointment, creates a Stripe deposit link, confirms via calendar. Zero staff.

**Architecture Flow:**
```text
Webhook → Normalize → Switch (Audio/Text)
  ├─ Audio → Base64 Decode → Gemini STT → Merge
  └─ Text ────────────────────────────────→ Merge
       → AI Agent (Gemini + Qwen3 Fallback)
           ├─ Tools: Calendar CRUD + Stripe + Pinecone RAG
           └─ Memory: Session-based Buffer
       → Switch (Audio/Text Response)
           ├─ Audio → Deepgram TTS → Respond
           └─ Text  → Format JSON  → Respond
