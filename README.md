> **The Problem:** Clinics lose massive revenue to missed calls and manual booking errors.
> **The Solution:** A full-stack AI SaaS. Speak or type in Egyptian Arabic — the AI Agent acts as "Dr. Samir", understands intent, creates a Stripe deposit link, and confirms via Google Calendar. Zero staff needed.

**⚙️ Architecture & Data Flow:**
```text
Client Webhook Intake → Custom JS 5-Layer Audio Detection
  ├─ Audio Stream → Base64 Decode → Gemini STT → Merge Context
  └─ Text Stream ──────────────────────────────→ Merge Context
       ↓
     AI Agent Engine (Gemini 2.5 Flash + Qwen3-Max Fallback)
       ├─ Tools invoked: Google Calendar (CRUD) + Stripe API + Pinecone RAG
       └─ Context: Session-based Buffer Window Memory
       ↓
     Response Router (Switch Node)
       ├─ If Audio → Deepgram API (TTS) → Respond via Voice
       └─ If Text  → Format JSON → Respond via Text
